HTML Tags Used & What They Do

<html>
The root tag. Everything on the page must be inside it.
<head>
Contains information about the page (title, etc.). You don't see it on the screen.
<title>
The text that appears in the browser tab or window title bar.
<body>
All the visible content goes inside this tag.
<h1> to <h6>
Headings. <h1> is the biggest, <h6> is the smallest. Use them for titles and sections.
<p>
Paragraph. Makes a block of text with nice spacing around it.
<br>
Line break. Forces text to go to the next line (used inside the quote above).
<hr>
Horizontal rule. Draws a line across the page to separate sections.
<a href="...">
Anchor / hyperlink. Makes text clickable so you can jump to another page or section.
<ul> + <li>
Unordered list + list item. Creates bullet point lists.
<img src="..." alt="...">
Embeds an image. src = where the picture is, alt = text shown if the image can't load.
<blockquote>
Used for quoting someone. Usually appears indented.
<dl> + <dt> + <dd>
Definition list + term + description. Perfect for making glossaries and explanations (like this list!).
<b>
Makes text bold.
<i>
Makes text italic.
